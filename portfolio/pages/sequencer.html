Creative Sequencer
In MaxMSP I created a MIDI note sequencing system that connects to Ableton and can be controlled using the Oxygen 25 controller.

I created a simple 16-step sequencer and added some complex features to it, trying to increase its unpredictability. The sequencer has a tempo-controlled timing system using GlobalTransport, a transposition system and probability-based systems.

The user can change parameters like the minimum and maximum values of the sequencer, the direction of the sequence, mode and pattern of a sequence jump. The symmetry and panning of the audio also can be modified. To introduce even more unpredictable outcomes, the user can add a possibility of a mute or a transposition.

I also used the presentation mode for my sequencer and properly commented each feature to ensure that even someone who is not familiar with this software can easily interact with the sequencer.
