Live Performance
I created a generative performance system in Ableton, using both looped audio recordings and MIDI instruments.

I built a harmonic melody in C Major using MIDI effects like Scale and generated clips, and added randomness with Ableton’s Generate tool and Expression Control. Clip behavior was shaped using Follow Actions to create evolving and unpredictable sequences. I also incorporated atmospheric audio, and warped it to fit the tempo. It was enhanced with effects like echo, panning, and automated envelopes. Then I added bass layers, percussion and an Operator synth structured with Follow Actions and audio effects to create evolving patterns.

For performance, I mapped key parameters to a MIDI controller. By adjusting these values with the controller, I changed the atmosphere throughout the song and created smoother transitions or introductions of instruments.
